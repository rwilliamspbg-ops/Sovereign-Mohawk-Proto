# Notified Body Early Engagement Checklist

Use this checklist before RC freeze when targeting regulated high-risk deployments.

## Trigger Criteria

- Healthcare/oncology-adjacent deployment target
- Geospatial critical-infrastructure use case
- High-risk classification likely under applicable framework

## Engagement Timing

Start outreach at architecture freeze or earlier, not after final RC.

## Pre-Read Package

- Intended purpose and use boundaries
- System architecture summary
- Risk management summary
- Verification/validation summary
- Post-market monitoring plan
- Incident and CAPA process summary

## Questions to Resolve Early

- Evidence format expectations
- Required depth for risk and traceability matrix
- Cybersecurity-specific dossier expectations
- Human oversight evidence expectations
- Post-market reporting expectations

## Internal Owners

- Compliance lead
- Engineering lead
- Security lead
- Release manager

## Outputs

- Meeting notes and action list
- Gap list against expected dossier format
- Updated TDF sections and closure plan
